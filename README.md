# BANKING-TRANSACTIONS-PROJECT

## Overview
This project analyzes retail banking transactions and loan portfolio data using Power BI. The goal is to understand customer behavior, assess loan exposure, and monitor credit risk through interactive dashboards.

## Datasets Used
- Retail Banking Transactions
- Banking Loans Portfolio

## Key Insights
- Transaction activity is dominated by high-value transactions, indicating strong customer engagement.
- Digital banking channels drive most transactions.
- Credit risk is concentrated in long-tenure loans.
- Some low-value customers hold high loan exposure.
- Longer-tenure loans demonstrate higher default risk compared to short-term loans.
- Repayment performance is stronger among lower-risk loan categories.
- A significant portion of the loan portfolio remains outstanding, increasing liquidity exposure.
